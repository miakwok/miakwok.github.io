github+hexo搭建个人博客

发表于 2018-10-04
字数统计: 729 | 阅读时长 ≈ 3

准备工作
1、注册开通github账号 记住账号和、密码以及验证邮箱
2、下载安装nodejs、npm
3、下载Git

1、创建github仓库
创建reopsitory 名称要设置为yourname.github.io 需要按照此规则创建

![repository][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/1.png]
![repository][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/2.png]

2、安装Git Bash
这里没什么技巧，就是下载安装 然后安装成功后可查看git version
![git version][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/3.png]

3、安装nodejs
搭建hexo博客是基于nodejs环境的静态博客~具体下载地址可在官网查找
安装完以后 需要在电脑环境变量配置，可在git bash查看node 是否安装成功
![node -v][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/4.png]

安装hexo
先创建一个文件夹（用于存放博客的文件） 然后cd到该文件夹下
安装hexo命令 $ npm install -g hexo-cli
（可查看官网文档：https://hexo.io/zh-cn/docs/）
安装完成后，可查看安装版本
![hexo -v][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/5.png]

初始化命令 hexo init 初始化完成后可看到文件夹下有以下文件
![hexoinit][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/6.png]

  • node_modules：是依赖包
  • public：存放的是生成的页面
  • scaffolds：命令生成文章等的模板
  • source：用命令创建的各种文章
  • themes：主题
  • _config.yml：整个博客的配置
  • db.json：source解析所得到的
  • package.json：项目所需模块项目的配置信息

搭桥到github
步骤一已经创建好repo,这时候需要回到git bash 配置github账户信息（yourname&email）

git config -global user.name “yourname”
git config -global user.email “youremail”
![hexoinit][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/7.png]
![hexoinit][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/8.png]

创建SSH
在gitbash中输入：ssh-keygen -t rsa -C youremail@example.com，生成ssh
然后找到id_rsa.pub文件的内容
![ssh][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/9.png]
将获取到的ssh放到github中

添加一个新的ssh key key就是获取的ssh title可随意
然后在gitbash 中验证是否添加成功 ssh -T git@github.com
![hexoinit][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/10.png]
![hexoinit][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/11.png]

5、修改博客配置文件
用编辑器打开博客项目，修改_config.yml文件（注意冒号之后有个半角空格）
修改deploy信息
![hexoinit][https://github.com/miakwok/miakwok.github.io/blob/master/images/pic/12.png]

回到gitbash 进入blog目录 执行以下命令
hexo clean
hexo generate(hexo g)
hexo server(hexo s)

打开浏览器输入http://localhost:4000 即可看到本地部署的博客了~~

6、上传到github
先安装 npm install hexo-deployer-git –save
然后按以下命令部署
hexo clean
hexo generate
hexo deploy

部署成功以后，打开http://yourname.github.io即可看到个人博客~
